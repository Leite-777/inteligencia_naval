Como conseguir uma chave API?

  • Acesse o Google AI Studio
  • Faça login com sua conta do Google, no canto superior direito da página.
  • Depois de fazer o login, visite a página de Chaves API
  • No canto superior direito da página, clique em Criar chave de API, e então pode apenas clicar em Criar Chave.
  • Na nova janela copie a chave de API, que se parece com uma sequência de caracteres aleatórios.

O que é uma chave API?

Uma chave de API é um código que permite você conversar e receber respostas de um serviço de IA, da mesma forma como faria no site do ChatGPT ou do Gemini.

A diferença é que você pode enviar mensagens de forma automatizada utilizando código, ao invés de escrever manualmente.

A chave será utilizada para o Gemini receber as posições do jogo de batalha naval, e enviar suas jogadas para o site.

Atualmente o Google disponibiliza o uso gratuito do Gemini, mas com um limite de usos que reinicia diariamente.

O que acontece com erros na Chave API?

O Jogo da Batalha Naval consegue funcionar mesmo quando o Gemini estiver indisponível, tanto por erros temporários quanto por utilizar a capacidade máxima diária.

Quando isso acontecer, o Piloto Automático realizará jogadas no seu lugar.

Não consegui a chave API, e agora?

A Inteligência Naval pode ser jogada sem uma Chave API, com o modo do Piloto Automático.

Note que o Piloto Automático é apenas um algoritmo, e não representa uma inteligência artificial verdadeira.

Se não foi possível obter sua Chave API no Google AI Studio, pressione o botão Jogar com Piloto Automático na página.

Jogo da Batalha Naval

Regras do Jogo

  • Cada jogador começa arrastando 5 navios para posições estratégicas do seu tabuleiro.
  • Na vez de cada jogador, ele deve escolher uma posição para atacar o tabuleiro do inimigo.
  • Caso essa posição acerte um navio, o mesmo jogador pode continuar atacando mais posições até errar uma.
  • Quando todas as partes de um navio são acertadas, ele é afundado.
  • Vence quem afundar os 5 navios do oponente.

    Sobre o jogo:

    A Inteligência Naval é um projeto que demonstra, na prática, como uma inteligência artificial pode ser integrada a um jogo complexo, como a Batalha Naval, utilizando a API do Gemini. Como modelos de linguagem não conseguem interagir diretamente com interfaces visuais, o sistema transforma o estado do jogo em dados estruturados no formato JSON e envia instruções detalhadas para que a IA analise o cenário e tome decisões a cada rodada.

    Com base nessas informações, a IA escolhe onde atacar e justifica sua decisão de forma estratégica, permitindo que suas ações sejam interpretadas e exibidas no site. O projeto também considera limitações técnicas, como possíveis falhas na API, implementando um sistema alternativo de jogadas automáticas para garantir a continuidade do jogo mesmo quando a IA estiver indisponível.
