Guia do Usuário

Uma chave de API é um código único que autentica o uso de um serviço de IA a receber comandos e enviar respostas.

O código será utilizado para o Gemini receber as posições do jogo de batalha naval, e enviar suas jogadas para o site.

  • Acesse o Google AI Studio
  • Faça login com sua conta do Google, no topo da página. Pode ser necessário verificar sua idade.
  • No topo da página, vá para Conferir a chave de API
  • Na nova página, no canto superior direito clique em Criar chave de API, e então em Criar Chave.
  • Na nova janela copie a chave de API, que se parece com uma sequência de caracteres aleatórios.
  • Cada jogador começa arrastando 5 navios para posições estratégicas da sua grelha.
  • Na vez de cada jogador, ele deve escolher uma posição para atacar a grade do inimigo.
  • Caso essa posição acerte um navio, o mesmo jogador pode continuar atacando mais posições até errar uma.
  • Quando todas as partes de um navio são acertadas, ele é afundado.
  • Vence quem afundar os 5 navios do oponente.
